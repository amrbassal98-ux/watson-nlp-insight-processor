# Final project

## Overview
This is an AI-based web application developed as a final project for the IBM Full-Stack Software Developer Professional Certificate. The application performs emotion detection on customer feedback using the Watson NLP library and is deployed using the Flask web framework.

## Features
- **Emotion Analysis**: Analyzes text for anger, disgust, fear, joy, and sadness.
- **Dominant Emotion Detection**: Identifies the strongest emotion expressed in the input.
- **Web Interface**: Allows users to input text and receive immediate emotional analytics.
- **Error Handling**: Gracefully handles blank inputs and server errors with appropriate user feedback.
